# 적용 방법

1. GitHub에 `datell1357` 이름의 Public 저장소를 만듭니다.
2. 이 ZIP의 내용을 저장소 루트에 그대로 업로드합니다.
3. 최종 구조:

```text
datell1357/
├── README.md
└── assets/
    ├── profile-terminal.svg
    └── cat-profile.png
```

`profile-terminal.svg`에는 고양이 이미지가 Base64로 내장되어 있으므로,
실제 표시에는 `cat-profile.png`가 없어도 됩니다. 다만 원본 보관 및 수정 편의를 위해 함께 포함했습니다.

Activity Graph와 GitHub Stats는 외부 공개 서비스가 GitHub 데이터를 읽어 동적으로 생성합니다.
해당 서비스 장애나 정책 변경 시 일시적으로 표시되지 않을 수 있습니다.
